# Teste Técnico – Desenvolvedor Frontend (EdTech)

Este projeto foi desenvolvido como parte de um teste técnico para a posição de Desenvolvedor Frontend. O objetivo foi criar uma página web fiel a um design do Figma, utilizando apenas tecnologias web puras.

## 🛠 Tecnologias Utilizadas

- **HTML5:** Estrutura semântica da página.
- **CSS3:** Estilização moderna com Flexbox e Grid, seguindo fielmente as cores e tipografia do design.
- **JavaScript Vanilla:** Lógica de interatividade, manipulação de DOM e persistência de dados.

## 📦 Funcionalidades Implementadas

1.  **Player de Vídeo:** Incorporação de vídeo do YouTube responsivo.
2.  **Slider de Imagens:** Implementado do zero com controles de navegação.
3.  **Player de Áudio:** Interface funcional utilizando a API nativa de áudio.
4.  **Atividades Interativas:**
    -   **Discursiva:** Com persistência no `sessionStorage` e estados de botões (Responder/Alterar).
    -   **Objetiva:** Seleção por checkbox com feedback visual e persistência.
5.  **FAQ (Accordion):** Implementação nativa com animações suaves de abertura e fechamento.
6.  **Responsividade:** Design adaptável para Desktop e Mobile.
7.  **Persistência:** Todos os dados das atividades são salvos e restaurados automaticamente ao atualizar a página.

## 🚀 Como Executar

1.  Extraia o conteúdo do arquivo `.zip`.
2.  Abra o arquivo `index.html` em qualquer navegador moderno.
3.  Não é necessário instalar dependências ou rodar servidores (embora um servidor local como o `Live Server` seja recomendado para uma melhor experiência).

## 🧠 Decisões Técnicas

-   Optei por não utilizar frameworks para demonstrar domínio sobre as APIs nativas do navegador.
-   O `sessionStorage` foi escolhido para a persistência conforme solicitado, garantindo que os dados permaneçam durante a sessão de navegação.
-   A estrutura de arquivos foi mantida simples e organizada para facilitar a leitura e manutenção.
